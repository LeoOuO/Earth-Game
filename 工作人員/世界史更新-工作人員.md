## 世界史更新流程（每次 U1–U4 依序執行）

1. **領地兵力結算**：依佔領矩陣規則判定各領地佔領隊伍，公告結果並給予兵力。
2. **特殊事件公告**：執行本週期對應的特殊事件（見下方時程）。

> ⚠️ 手上旗子加扣兵力機制已移除，每次世界史更新**僅**結算領地佔領兵力。

---

## 特殊事件時程

| 時間 | 事件 | 類型 | 效果說明 |
| :---- | :---- | :---- | :---- |
| **13:00** | 活動開局 | 初始化 | 講解規則、分組（1a/1b）出發；各領地開放第一輪挑戰 |
| **U1 13:30** | 開港通商 | 市集開放 | 中央市集正式開放 |
| | 補給空投 | 經濟 | 目前累積兵力最低兩個小隊各獲得 100 海石 |
| | 掏金熱 | 領地 | 隨機一個領地成為「黃金領地」，下次更新前佔領得 200% 基礎兵力 |
| **U2 13:55** | 物價波動 | 市集 | 限時大波動：指定選卡與隨機抽卡價格全部減半 |
| | 邊境封鎖 | 空間 | 隨機封鎖 2 個領地，期間不能插入新旗子（原旗不受影響，關卡仍可挑戰）|
| | 情報洩漏 | 諜報 | 大螢幕公布每個小隊手上的旗子總數與海石數量 |
| **U3 14:20** | 戰場迷霧 | 資訊 | 世界核心只公布領地旗子總數與佔領狀態，不公布是哪隊佔領 |
| | 情報交換 | 諜報 | 各組可向隊輔詢問一項其他隊伍的資訊，14:40 一次統一公告 |
| | COW 說明書公告 | 資訊 | 公布 COCONUT WARS 說明書連結，小隊員可開始了解最終大戰規則 |
| **U4 14:45** | 旗幟徵召 | 強制 | 所有小隊必須在 15:10 前將手中旗子全部插完 |
| | 大規模轟炸 | 強制 | 隨機四個領地遭受轟炸，各炸掉 1/3 面旗子 |
| **15:10** | COCONUT WARS（COW）| 決戰 | 結算資源，進入 COCONUT WARS |
| **15:40** | 終極審判 | 結算 | 最終總積分計算，執行懲罰與獎勵 |
