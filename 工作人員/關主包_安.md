<a id="關卡一布丁狗族碰撞不停狗"></a>
## 關卡一：布丁狗族 — 碰撞不停狗

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 布丁狗族 |
| **需要道具** | 棋盤（貼膠帶在地上）、不同顏色的「不停狗」棋子（拿紙板簡單做然後印不停狗貼上去）、分光鏡道具（拿紙板簡單做然後印分光鏡貼上去）、題目卡 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **衝撞面積**：上下左右相鄰同色旗子的**最大連通面積**最大的隊伍佔領。|
| **適合人數** | 單組4～8人，最多兩組競爭 |
| **建議時間** | 每回合2分鐘，共1回合試玩+3回合 |

### 故事背景


歡迎來到布丁狗國！這個種族全身充滿彈性、熱愛奔跑，但因為身體太圓了，一旦起跑就完全無法自行煞車，必須撞到牆壁或其他同伴、或是遇到「分光鏡」折射才能強制轉彎。今天布丁狗要在錯綜複雜的迷宮中尋找聖光出口，挑戰者必須擔任「導航 AI」，幫停不下來的布丁狗找出最精準的移動路徑。


### 遊戲規則

- 不停狗要撞到牆壁，或是撞到其他不停狗才能停下來；如果撞到分光鏡，要照著分光鏡的方向反射過去。
- 每次開始前先抽題目，指定哪一個顏色的不停狗最終要到哪一個位置，看誰能最快找到路徑。
- 主要流程：選出四個人當機器人（上一輪沒玩到的，下一輪一定要上來）→ 抽題目  →  場上的不停狗開始移動（場下的其他人可以出聲指引）。
- 遊戲總共進行三回合。

### 遊戲變化

**單組**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 若比關主想到的步數多 3 步以內，就算成功 |
| 計分 | 每比目標步數少一步，回合得分加 1（供關主記錄排名用）|

**多組競爭**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 看誰比較快走到終點，三回合依累積分數排名（第一名 5 分、第二名 3 分、第三名 1 分）|

### 遊戲獎勵

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 海石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功（三回合合格） | 3 本 ＋ 1 他 | 25 |
| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 15 |
| 多組競爭 | 第 1 名 | 4 本 ＋ 1 他 | 35 |
| 多組競爭 | 第 2 名 | 3 本 ＋ 1 他 | 25 |
| 多組競爭 | 其餘 | 2 本 ＋ 0 他 | 15 |

### 特別備註

- 關主需事先計算好每道題目的最少步數（解答）並記錄在題目卡背面，供現場對照。
- 每當有人要移動時，要先舉手，等關主喊出紅色（若他代表紅色布丁狗），他才能移動，以避免一次兩個人移動造成混換．
- 棋盤起始狀態需依題目卡準確擺放後，才讓隊員上棋盤，避免誤操作造成爭議。

<a id="關卡二河童國接水管接力"></a>
## 關卡二：河童國 — 接水管接力

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 河童國 |
| **需要道具** | 水管零件（剪不同形狀的紙板）、膠帶（貼組裝區）、印題目工程圖（藍圖）、起點/中點/終點標記（膠帶） |
| **需要關主人數** | 2 人（1 人在中點、1 人在終點）|
| **佔領規則** | **水路貫通**：從第 0 列連通到第 3 列的隊伍佔領（旗子需左右相鄰連通）。無人達成則比旗子總數；多隊達成則比最少格子數連通者佔領；仍平手先到先得。|
| **適合人數** | 單組4到10人，多組競爭各組4到10人 |
| **建議時間** | 共三回合；單組限時（例如 90 秒）完成 |

### 故事背景

來到幻境大陸的神秘流域「河童國」。河童族的生活片刻都離不開水，牠們靠著盤根錯節的水管將聖泉引導至村莊。然而，沼澤遭遇怪獸襲擊，導致供水管線全數斷裂。這是一場考驗團隊速度、空間記憶與精準度的接力賽，挑戰者必須一棒接一棒，在混亂的零件堆中看清工程圖、抓對水管，合力將引水導管重新連通，拯救面臨乾枯危機的河童族！

### 遊戲規則

裁判鳴哨後，各組隊員以接力賽跑方式進行：

1. 每棒全速衝刺至中點查看地圖進度。
2. 再奔跑至終點抓取長度與形狀相符的水管零件。
3. 返回中點將零件組裝於地墊上。
4. 最後折返起點與下一棒擊掌。
5. 台下等待的隊員可以在起點指揮。
6. 當整條導管成功連通起點與終點，即完成通關。

共三回合。

### 遊戲變化

**單組**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 在指定時間內（例如 90 秒）完成 |

**多組**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 每組一個跑道，看哪一組先完成 |

### 遊戲獎勵

> 率先完成接管的組別趟過神秘流域，帶回的寶藏中夾帶其他王國旗子。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 海石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功（時限內完成） | 3 本 ＋ 2 他 | 30 |
| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 15 |
| 多組競爭 | 第 1 名 | 4 本 ＋ 2 他 | 35 |
| 多組競爭 | 第 2 名 | 3 本 ＋ 1 他 | 25 |
| 多組競爭 | 其餘 | 2 本 ＋ 0 他 | 15 |

### 特別備註

- 關主 A 負責中點：確認選手查看藍圖方式（只能用眼睛看，不能拍照或抄寫），管理組裝地墊上的水管狀態。
- 關主 B 負責終點：管理水管零件堆，確認選手取走的零件種類正確，防止選手一次抱走大量零件。
- 每回合開始前需將藍圖換為新的那回合題目（建議用 A4 裝入文件套）。
- 若隊員在中點「組裝」時零件對不上，關主 A 需提醒但不指導，讓隊員自行解決或返回終點換件。

<a id="關卡三哥布林族地底金庫的歌不停"></a>
## 關卡三：哥布林族 — 地底金庫的歌不停

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 哥布林族 |
| **需要道具** | 假鈔數疊（各疊張數不同）、音樂播放設備（洗腦神曲） |
| **需要關主人數** | 1 人 |
| **佔領規則** | **金庫方塊**：完整 **2×2 旗子方塊**（4 格全為同色）數量最多的隊伍佔領（可重疊計算，最多 9 個）。|
| **適合人數** | 每次一小隊派出 4 人 |
| **建議時間** | 一回合兩分鐘（1分鐘遊戲+1分鐘對答案），共四回合(1回試玩+3回正式遊戲) |

### 故事背景

來到貪財且熱愛狂歡的「哥布林族」地底領地。這裡的金庫深處永遠播放著節奏極強的洗腦神曲，而哥布林們最強的天賦，就是能一邊大聲唱歌、雙手還能一邊精準數錢。人類想要帶走這裡的寶藏，就必須通過這場「一心多用」的數鈔考驗——在嘴巴被迫唱出歌詞、大腦被旋律轟炸的同時，僅憑觸覺盲數出正確的鈔票張數！

### 遊戲規則

- 挑戰者手裡拿著一疊假鈔進行盲數。
- 背景播放節奏極強的洗腦神曲。
- 挑戰者數錢時**嘴巴絕對禁止數出數字，必須跟著旋律唱出歌詞**。
- 一次一小隊派出四個人，四個人的鈔票數目會不同，加總後看離正確的結果多近。

### 遊戲變化

**單組**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 4 人的總誤差在 5 張以內算成功 |
| 計分 | 誤差少於 3 張 +1 分；完全正確 +2 分（供關主記錄）|

**多組**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 看哪一組最接近正確數量，第一名 5 分、第二名 3 分、第三名 1 分 |

### 遊戲獎勵

> 金庫裡混著其他王國的財寶，成功數清楚的組別能帶走其他國旗子。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 海石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功（誤差 ≤5）| 3 本 ＋ 1 他 | 20 |
| 單組闖關 | 失敗（誤差 ＞5）| 2 本 ＋ 0 他 | 10 |
| 多組競爭 | 第 1 名 | 4 本 ＋ 1 他 | 25 |
| 多組競爭 | 第 2 名 | 3 本 ＋ 0 他 | 15 |
| 多組競爭 | 其餘 | 2 本 ＋ 0 他 | 10 |

### 特別備註

- 假鈔的實際張數應事先數好並封存在信封中，挑戰開始前才打開讓選手拿取；每回合結束後由關主清點確認。
- 歌曲節奏越強越好，建議選副歌重複、朗朗上口的歌；全程大聲播放，若有耳機或高音量喇叭效果更佳。
- 關主的「正確答案」（每疊幾張）需記在自己的小紙條上，不公開，等四人數完後對答案。
- 歌曲：
--試玩：生日快樂歌、小星星
--第一輪（兒歌）：卡加布列島、釣魚記、小毛驢
--第二輪（正常的歌）：妳是我的花朵（伍佰、可以快轉到副歌）、捲煙、擱淺
--第三輪（干擾性較強的歌）：rickroll、喔喔喔愛

<a id="關卡四套娃族俄囉斯套娃版圈圈叉叉"></a>
## 關卡四：套娃族 — 俄囉斯套娃版圈圈叉叉

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 套娃族 |
| **需要道具** | 大、中、小尺寸塑膠杯各 2 個 × 每隊（兩種顏色），九宮格桌或地墊 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **嵌套對角**：斜向（`\` 或 `/` 方向）**最長連線**最長的隊伍佔領。平手以旗子總數決。|
| **適合人數** | 兩組競爭 |
| **建議時間** | 一回合2分鐘，共4回合（1回試玩+3回正式） |

### 故事背景


來到幻境大陸最神秘的「套娃族」聚落。套娃族相信萬物皆可嵌套、大者恆包小者——這是他們的生存哲學，也是他們的作戰智慧。族中長老以無數層套疊的魔法聖杯守護著九宮格圖騰，大杯能直接吞噬對手的小杯，只要你夠大，你就是法則！挑戰者必須化身為套娃族的戰士，以最快的速度讓己方顏色連成一線，才能打破封印、奪下這塊傳說中「大的永遠吃小的」神秘領地！

### 遊戲規則

- 每隊擁有大、中、小尺寸各 2 個塑膠杯。
- **大杯可直接覆蓋並吃掉對手的小杯**。
- 小隊員輪流衝到九宮格放杯子，依序先從小的放到大的。
- 棋子放完後，後續棒次須衝至桌前「移動」場上己方棋子。
- 率先使己方顏色連成一線（橫、豎、斜）即獲勝。

### 遊戲變化

**兩組競爭**

| 項目 | 說明 |
| :---- | :---- |
| 目標 | 看哪一組贏比較多回合 |

### 遊戲獎勵

> 被大杯吞掉的一方往往留下敵方標記（他國旗子）。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 海石 |
| :---- | :---- | :---- | :---- |
| 兩組競爭 | 勝隊 | 3 本 ＋ 1 他 | 25 |
| 兩組競爭 | 敗隊 | 1 本 ＋ 1 他 | 10 |
| 兩組競爭 | 平手（各隊） | 2 本 ＋ 0 他 | 15 |

### 特別備註

- 「大杯覆蓋小杯」的規則要在遊戲開始前演示一遍，確保所有選手都理解。
- 杯子放置需在九宮格的格子正中央，若放歪或壓線請讓選手重新放置再出發。
- 移動己方棋子時，不得直接抱走對手的小杯移到新位置——只能用己方大杯「覆蓋」留在原格，無法帶著走。
- 小紙杯：5.5oz 直徑6 高7.5 , 中紙杯：9oz 直徑7.5 高9 , 大紙杯：16oz 直徑9 高13（各需4個）
