<a id="關卡一人類王國九宮戰術陣"></a>
## 關卡一：人類王國 — 九宮戰術陣

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 人類王國 |
| **需要道具** | 九宮格地墊或膠帶 1 組、兩色隊伍棋子各 3 個、計時器、隊伍記分表 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **戰術連線**：橫、直、斜任一方向中擁有**最長連線**的隊伍佔領。平手以連線條數多者決；仍平手先到先得。|
| **適合人數** | 單組 4–10 人；兩組競爭各 4–10 人；多組可採輪戰或同時開兩個九宮格 |
| **建議時間** | 講解 1 分鐘；每局 2 分鐘；換場與記分 1 分鐘 |
| **場地配置** | 地上畫 3×3 九宮格；兩隊起跑線離九宮格約 3–5 公尺，避免衝撞 |

### 故事背景

在人類王國的王都，騎士團以九宮戰術陣挑選能進入城門的隊伍。每一次猜拳代表一次短兵交鋒，贏的人才能搶下一格陣地；隊伍必須在有限時間內完成三格連線，證明自己有指揮與臨場判斷能力。

### 場地與準備

- 兩隊各拿同色棋子 3 個，隊員在起跑線後排隊；單組模式則由關主代表「王國守衛」對戰。
- 關主先示範「猜拳、放棋、回隊伍拍下一位」的流程，確認大家知道不能推擠。
- 多組模式若只有一個九宮格，採擂台輪戰：勝隊留場，下一隊挑戰；每隊至少打一局。

### 遊戲規則

1. 每回合由雙方各派一人跑到九宮格前猜拳，贏者可以放 1 個己方棋子；平手重猜一次，第二次仍平手則雙方都不能放棋，直接換下一位。
2. 每隊最多同時放 3 個棋子；當第 4 次贏得放棋權時，改成移動己方任一棋子到空格。
3. 任一隊先完成橫、直、斜任一條三連線即獲勝。
4. 單組模式：隊伍與關主對戰，隊伍在 2 分鐘內完成三連線即成功；若關主先連線或時間到未連線則失敗。
5. 多組輪戰：每局勝隊得 2 分，平手各 1 分，全部輪完依總分排名；同分以「最快完成連線」排名。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 隊伍對上關主（王國守衛），2 分鐘內三連線即成功 |
| 兩組競爭 | 兩隊同時在場上 PK，先三連線者勝 |
| 多組輪戰 | 勝隊留場，每隊至少打一局，依積分或連線時間排名 |

### 遊戲獎勵

| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 12 |
| 兩組競爭 | 勝隊 | 3 本 ＋ 2 他 | 28 |
| 兩組競爭 | 敗隊 | 1 本 ＋ 1 他 | 10 |
| 兩組競爭 | 平手（各隊） | 2 本 ＋ 1 他 | 18 |

| 多組競爭 | 其餘 | 1 本 ＋ 0 他 | 10 |

**Bonus（星石）**
- 「軍師 Bonus」：開始前 10 秒討論策略，且每位隊員至少上場一次並獲勝，全隊 **＋2 星石**。
- 「逆轉 Bonus」：落後兩格後逆轉勝，額外 **＋2 星石**。

### 特別備註（關主注意事項）

- 棋子要放在格子中央，壓線時由關主裁定。
- 跑動只允許快走或小跑，場地滑時改成「走到九宮格」避免跌倒。
- 若隊伍人數差很多，少人的隊伍可以有人重複上場，但需照固定順序。
- **違規推擠、故意擋人或拿錯棋子：第一次警告，第二次該回合放棋權作廢。**

<a id="關卡二精靈森域禁語回聲林"></a>
## 關卡二：精靈森域 — 禁語回聲林

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 精靈森域 |
| **需要道具** | 題目卡 20–30 張、計時器、記分板、分組色貼或號碼牌 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **以少勝多（✦特殊規則）**：場面上己方旗子**最少**（至少 1 面）的隊伍佔領。插旗階段可用己方旗子 1:1 替換場面上任意一面旗子（含己方），被換下旗子回到持有者手中。並列最少時以旗子總數決；仍平手先到先得。|
| **適合人數** | 單組 5–10 人；兩組合作 10–18 人；多組競爭 3–5 組皆可 |
| **建議時間** | 講解 1 分鐘；正式 2–3 輪約 5 分鐘；Bonus 1 分鐘 |
| **禁用方式** | 禁止講話、口型、寫字、直接比數字或注音；只能用肢體動作與表情 |

### 故事背景

精靈森域受到沉默結界保護，進入森林的人不能說話，只能靠動作傳遞訊息。每組都是一支來求取通行證的使節團，必須讓訊息在混亂中被理解，才有資格穿越森林。

### 場地與準備

- 單組模式：同組排成一列，第一位看題目後依序傳動作，最後一位回答。
- 多組模式：各組派出同樣人數排成並列縱隊；每組第一位看不同題目，同時開始傳。
- 若想做「跨組傳遞」版本，將隊伍依 A-B-C-A-B-C 排成大圈，每輪傳給右手邊下一位；猜中者所屬隊伍與出題隊伍都得分。

### 遊戲規則

1. 第一位看題目 5 秒後開始表演給下一位看，每次傳遞最多 10 秒。
2. 看完動作者不能回問、不能討論，只能把自己理解的內容繼續演給下一位。
3. 最後一位有 10 秒回答，可答一次；答案意思接近即可，由關主判定。
4. 多組競爭採 3 輪，每輪換一張題目；每組「最後猜對」得 2 分，「中途有人明顯猜對關鍵字」可得 1 分。
5. 跨組版本計分：成功讓別組猜到自己的題目，出題隊 +1；猜中別組題目，猜題隊 +1，因此總分 = 猜對數＋被猜對數。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 同組排成一列傳遞，3 題中至少 2 題最後答案正確即成功 |
| 兩組合作 | 兩組接成一條長隊，最後答案正確即兩組共同成功 |
| 多組競爭 | 各組並列縱隊，輪數結束後依總分排名；同分比答題速度 |
| 跨組傳遞 | 全員圍大圈，每輪傳給右手邊，猜中算雙方得分 |

### 遊戲獎勵

> 兩組合作模式下，兩組各自獲得同等獎勵。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功 | 2 本 ＋ 1 他 | 18 |
| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 10 |
| 兩組合作 | 成功（各組） | 3 本 ＋ 1 他 | 22 |
| 兩組合作 | 失敗（各組） | 2 本 ＋ 1 他 | 18 |
| 多組競爭 | 第 1 名 | 3 本 ＋ 1 他 | 22 |
| 多組競爭 | 第 2 名 | 2 本 ＋ 0 他 | 14 |
| 多組競爭 | 其餘 | 1 本 ＋ 1 他 | 10 |

**Bonus（星石）**
- 「古精靈語 Bonus」：最後一輪改成抽象詞，答對的隊伍 **＋3 星石**。
- 「無聲完美 Bonus」：整輪無出聲、口型或寫字，該組 **＋1 星石**。

### 特別備註（關主注意事項）

- 題目難度要混合：食物、動物、職業、電影角色、校園物品各準備一些。
- 關主判定「意思接近」即可，不要卡太細（如「電腦」和「筆電」可視題目決定）。
- 若隊伍人數不同，讓人數多的隊伍輪空幾位，保持每組傳遞人數一致。
- **出現說話、口型或手寫提示：該題 0 分；第二次違規可扣 1 分。**

## 關卡三：龍族火山 — 龍王試煉場

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 龍族火山 |
| **需要道具** | Boss 血量板（建議白板畫愛心）、血量磁鐵或紙條、計時器、Bonus 卡 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **群礁稱王（✦特殊矩陣 6×6）**：使用 6×6 矩陣（36 格）。計算各隊旗子形成的連通區域數（上下左右相鄰為連通），區域數最多者佔領。並列時以旗子總數決；仍平手先到先得。|
| **適合人數** | 單組 4–10 人；多組合作 2–4 組可一起打 Boss |
| **建議時間** | 講解 1 分鐘；正式挑戰 3 分鐘；結算 30 秒 |
| **核心難度** | 只有猜贏才算成功；平手與猜輸皆視為挑戰失敗。造成傷害需要「連續猜贏兩次」 |

### 故事背景

龍族火山口住著守門龍王，牠不靠蠻力決定通行資格，而是用古老的猜拳儀式測試勇者的膽量。每位挑戰者都要連續看破龍王兩次出拳，才能打碎一層龍鱗護盾。

### 場地與準備

- 單組模式：Boss 初始 HP = 3；隊伍排隊依序挑戰關主。
- 多組合作：所有隊伍共同打同一隻 Boss，Boss HP = 參與組數 × 2 + 2（例如 3 組時 HP = 8）。
- 每次挑戰結束後立刻換下一位，避免同一人一直猜。多組合作時依 A-B-C-A-B-C 順序輪流派人。

### 遊戲規則

1. 挑戰者必須連續猜贏關主兩次，才對 Boss 造成 1 點傷害。
2. 若第一拳或第二拳平手／猜輸，該次挑戰失敗，直接換下一位。
3. 限時 3 分鐘內將 Boss HP 歸零即闖關成功。
4. 每位玩家至少要出場一次；若人數太多，關主可以縮短每次猜拳間隔並用倒數加速。
5. 多組合作仍記錄各組造成的傷害數，作為 Bonus 與貢獻獎勵依據，但主要目標是共同打倒 Boss。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | Boss HP = 3，3 分鐘內打倒 |
| 多組合作 | Boss HP = 組數 × 2 + 2，所有組輪流出人打同一隻王 |
| 多組改競爭（備案） | 各組獨立對關主，計 3 分鐘內造成的傷害數排名 |

### 遊戲獎勵

> 合作模式中各組各自領取同等獎勵。龍王火山的寶藏中夾帶其他王國旗幟，成功討伐者可獲得更多他國旗子。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功 | 3 本 ＋ 1 他 | 28 |
| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 14 |
| 多組合作 | 成功（各組） | 4 本 ＋ 2 他 | 35 |
| 多組合作 | 失敗（各組） | 3 本 ＋ 1 他 | 28 |
| 多組競爭備案 | 第 1 名 | 3 本 ＋ 2 他 | 30 |
| 多組競爭備案 | 第 2 名 | 3 本 ＋ 1 他 | 22 |
| 多組競爭備案 | 其餘 | 2 本 ＋ 0 他 | 14 |

**Bonus（星石）**
- 「龍焰暴擊」：同一玩家連續猜贏三次，該組 **＋2 星石**。
- 「全員勇者」：所有人至少上場一次並打倒 Boss，全組 **＋2 星石**。

### 特別備註（關主注意事項）

- 關主出拳要自然，不要讓玩家覺得故意讀心或故意放水。
- 多組合作時，關主需要大聲報 Boss 剩餘 HP，讓氣氛比較燃。
- 如果隊伍太小或運氣太差，可以中途啟用「火山祝福」：HP 減 1 或改成連贏一次造成半點，避免卡太久。

## 關卡四：獸人荒原 — 荒原躲避戰

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 獸人荒原 |
| **需要道具** | 軟式躲避球 1–2 顆、角錐或膠帶邊界、隊伍色帶、計時器 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **潮汐均勢（✦特殊規則）**：世界史更新時先計算全場旗子總數；總數為奇數則本輪無人佔領（領地中立）；總數為偶數則旗子最多的隊伍佔領。並列先到先得。|
| **適合人數** | 單組 5–10 人；兩組競爭各 5–10 人；多組競爭 3 組可採混戰或輪戰 |
| **建議時間** | 安全提醒 1 分鐘；正式每局 3 分鐘；決鬥或結算 1 分鐘 |
| **安全規則** | 禁止瞄頭、禁止高速近距離砸人、禁止推擠；頭部被打不算出局，投擲方警告 |

### 故事背景

獸人荒原的部族不靠口頭辯論決定通行權，而是用古老的躲避戰分出勇士。能在荒原球戰中保護隊友、閃避攻擊並留到最後的隊伍，才會獲得獸人部族的戰旗。

### 場地與準備

- 兩組競爭：場地分成左右兩半，各隊在自己的半場活動。
- 多組競爭：若場地夠大，分成 3–4 個區域混戰；若場地不夠，採輪戰積分制。
- 單組模式：改為「獸人訓練場」，隊伍派 2–3 位閃避者，其他人或關主在安全距離外投球，挑戰 60 秒存活。

### 遊戲規則

1. 正式局限時 3 分鐘，開始時球放中線或由關主拋球。
2. 球直接擊中身體且未先落地，該玩家出局；若被接住，投球者出局，接球隊可復活 1 位最早出局的隊友。
3. 球落地後再碰到人不算；碰到頭部不算，且投球方被警告。
4. 多組混戰時，可以攻擊任一隊，但不能跨出自己的區域太多；出界接球或投球皆無效。
5. 單組訓練場：60 秒內保留至少一半閃避者即成功；若全員存活視為完美成功。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組訓練場 | 派 2–3 閃避者，關主投球，60 秒存活過半即成功 |
| 兩組競爭 | 左右半場對打，3 分鐘比剩餘人數 |
| 多組混戰 | 各組佔一區域，可攻擊任意隊，依出局順序排名 |

### 遊戲獎勵

> 戰敗或失敗時，敵方旗幟散落在戰場上，因此敗者會拿到他國旗子。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
