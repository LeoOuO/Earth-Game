# 最終大戰 WWW — 操作手冊

最終大戰（WWW）是 NTU CSIE Camp 大地遊戲的收尾戰鬥階段（15:10–15:45），透過本網頁程式即時輸入各隊指令、執行回合結算、並以視覺化地圖呈現戰況。

---

## 一、環境需求

| 項目 | 最低版本 |
| :--- | :--- |
| Python | 3.10+ |
| 瀏覽器 | Chrome / Firefox / Safari（現代版本） |
| 作業系統 | macOS / Linux / Windows（WSL） |

---

## 二、下載與啟動

### 1. 下載 repo

```bash
git clone <repo-url>
cd <repo>/www
```

### 2. 啟動伺服器

```bash
./start.sh           # 預設 port 5001
./start.sh 8080      # 自訂 port
```

> **Windows 用戶**：請在 WSL 或 Git Bash 執行，或直接執行：
> ```bash
> python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt
> python app.py
> ```

第一次執行會自動建立虛擬環境並安裝相依套件。

### 3. 開啟介面

瀏覽器前往 `http://localhost:5001`（或指定的 port）。

## 三、賽前設定

首次進入會看到「賽前設定」畫面，共三個區塊：

### 3-1 基本設定

- **回合數**：填 1–5，遊戲規則為 3 回合。
- **參賽隊伍**：勾選本場參賽的隊伍編號（預設全選 1–10）。

### 3-2 初始領地設定

大地遊戲結束時（15:10），哪支隊伍佔領了哪個領地，就在對應的欄位填入：

```
格式：隊伍號:兵力，多隊共存用逗號分隔
例：1:500        → 隊 1 以 500 兵佔領
例：1:300,2:200  → 隊 1 有 300 兵、隊 2 有 200 兵共存
```

留空表示該領地無人佔領（空島）。

## 四、主畫面操作

### 4-1 整體佈局

```
┌─────────────────────────────────────────────────────┐
│  Header：回合資訊 ｜ 上/下一回合 ｜ 真實模式 ｜ 執行  │
├──────────────────────────┬──────────────────────────┤
│                          │  指令輸入區              │
│  地圖（分析/真實模式）     │  ─────────────────────  │
│                          │  本回合指令記錄           │
├──────────────────────────┴──────────────────────────┤
│  國力排行                                            │
└─────────────────────────────────────────────────────┘
```

### 4-2 輸入指令

1. 從左上「**輸入隊伍**」下拉選單選擇要輸入的隊伍。
2. 點擊指令模板按鈕（moving / attack / union / union_attack / set）可快速插入指令。
3. 在文字框輸入一至三條指令，每條可用**換行**或**空格**分隔，參數之間的空格不影響解析：
   ```
   moving(HK, ELF, 500)
   attack(ELF, ORC, 300)
   ```
4. 按「**發送**」提交，指令會顯示在下方記錄中並即時標示有效 ✓ / 無效 ✗。
5. 提交後可對每條指令點「**✏**」編輯（回填到文字框）或「**✕**」刪除，不需重新輸入全部。

### 4-3 指令格式

> 所有區域參數可用**英文代號**（不分大小寫）或**中文全名**。

#### moving — 移動兵力

```
moving(S, E, n)
```

| 參數 | 說明 |
| :--- | :--- |
| S | 出發區域（己方領地 / 中立 / 資源點） |
| E | 目的區域（同上限制，不可移入他國領地） |
| n | 兵力數（正整數） |

```
moving(HK, ELF, 500)
moving(NEU, HK, 200)
```

#### attack — 進攻

```
attack(S, E, n)
```

| 參數 | 說明 |
| :--- | :--- |
| S | 出發區域（己方控制） |
| E | 目標（必須為他國領地，不可攻中立小島） |
| n | 派遣兵力數 |

```
attack(HK, ORC, 400)
```

#### union — 聯盟（防禦型兵力轉移）

```
union(S, P, E, n)
```

| 參數 | 說明 |
| :--- | :--- |
| S | 出發區域 |
| P | 聯盟夥伴隊伍編號 |
| E | 目標領地（E 為 P 的領地時為請求方；E 為自己的領地時為接受方） |
| n | 兵力數 |

**需雙方同時提交**，且 S、E、n 完全相同、P 互相指名：

```
# 隊 1 請求：
union(HK, 2, ELF, 200)   ← 隊 1 從 HK 轉移 200 兵到隊 2 的 ELF

# 隊 2 接受：
union(HK, 1, ELF, 200)   ← 隊 2 同意
```

#### union_attack — 聯合進攻

```
union_attack(S, P, E, n)
```

| 參數 | 說明 |
| :--- | :--- |
| S | 出發區域 |
| P | 聯合進攻夥伴，多人用 `[1,2]` 格式 |
| E | 攻擊目標（他國領地） |
| n | 己方派遣兵力數 |

以**最大互信優先**配對：所有成員都互相指名才構成有效聯盟；若有衝突則取兵力總和最大的那組：

```
# 三國完整聯盟：
union_attack(HK, [2,3], ORC, 300)   # 隊 1
union_attack(ELF, [1,3], ORC, 200)  # 隊 2
union_attack(DV, [1,2], ORC, 100)   # 隊 3
```

#### set — 管理員直接設定領地（選擇「管理員」帳號使用）

```
set(zone, T:n, T:n, ...)   # 設定領地兵力
set(zone)                   # 清空領地
```

```
set(HK, 1:500, 2:200)
set(ORC)
```

### 4-4 執行回合

全部隊伍輸入完畢後，按右上角「**▶ 開始執行**」。

- 系統會同步執行所有指令（移動、進攻、聯盟）並計算結算（戰鬥、國力、中立增長）。
- 結果彈窗會顯示本回合完整 log。
- **真實模式**下會有船艦移動 + 戰鬥特效動畫。

### 4-5 回合導航

| 按鈕 | 說明 |
| :--- | :--- |
| ↩ 上一回合 | 撤銷已執行的回合，回到上一個狀態 |
| ↪ 下一回合 | 使用「上一回合」後才可用，往前還原 |

### 4-6 顯示模式切換

點選右上角「**🗺 真實模式**」可切換：

- **分析模式**（預設）：表格式領地檢視，適合確認數值。
- **真實模式**：海洋地圖，島嶼位置分佈，執行時有動畫效果。

## 五、常見問題

**Q：指令顯示 ✗ 是什麼意思？**
指令解析或驗證失敗。看指令下方的錯誤提示，常見原因：兵力不足、出發區域不是己方控制、超過每回合 3 次上限。

**Q：union 顯示「等待確認」？**
代表對方尚未提交對應的 union 指令。兩方都提交後會自動更新為「聯盟成立」。

**Q：Port 被佔用無法啟動？**
換用其他 port：`./start.sh 8080`

**Q：想重設遊戲怎麼辦？**
按右上角「⚙ 重新設定」回到賽前設定畫面。
