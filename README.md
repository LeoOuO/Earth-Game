# 大地遊戲（Earth Game）

NTU CSIE Camp 主活動「大地遊戲」的完整設計文件庫。

## 遊戲概述

- **時間**：13:00–16:00
- **規模**：10 小隊 × 約 10 人
- **地點**：一樓各教室（大地遊戲）+ 103（世界核心 & 最終大戰）

遊戲分為兩大階段：
1. **大地遊戲**（13:00–15:10）：各小隊分組探索 12 個領地關卡，透過四次世界史更新累積兵力。
2. **最終大戰 WWW**（15:10–15:45）：以累積兵力為初始資源，進行三回合的領地攻防，以國力決定最終排名。

## 核心機制

- **兵力系統**：每次世界史更新（U1–U4）由領地佔領（★ 500～★★★★★ 1000 兵/次）與手中旗子（己方 +30、他隊 −50）累積兵力，直接帶入最終大戰。
- **12 領地**：各有獨立 4×4 佔領規則，難度與兵力獎勵掛鉤（見[佔領矩陣](佔領矩陣.md)）。
- **道具卡**：22 種 49 張（N/R/SR/SSR），透過中央市集 103 購買或抽取。
