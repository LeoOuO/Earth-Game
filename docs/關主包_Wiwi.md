<a id="關卡一人類王國九宮投陣"></a>
## 關卡一：人類王國 — 九宮投陣

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 人類王國 |
| **需要道具** | 九宮格地墊或膠帶 1 組、兩色隊伍沙包各 3 個、隊伍記分表 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **戰術連線**：橫、直、斜任一方向中擁有**最長連線**的隊伍佔領。平手以連線條數多者決；仍平手先到先得。|
| **適合人數** | 單組 4–10 人；兩組競爭各 4–10 人 |
| **建議時間** | 講解 1 分鐘；每局 2 分鐘；換場與記分 1 分鐘 |
| **場地配置** | 地上畫 3×3 九宮格；兩隊起跑線離九宮格約 3–5 公尺，避免衝撞 |

### 故事背景

在人類王國的王都，騎士團以「九宮投陣」挑選能進入城門的隊伍。九宮格中的每一格都代表一處重要陣地，挑戰者必須奔赴前線，將象徵軍隊的沙包精準投入格中。率先占領橫、直或斜三處相連陣地的隊伍，便能突破王國守衛的防線，證明自己的速度、準度與戰術能力。

### 場地與準備

* 在場地中央以巧拼、膠帶或呼拉圈設置一個 **3×3 九宮格**。
* 在九宮格前方約 2.5～4 公尺處設置投擲線
* 投擲線後方再設置起跑線，兩隊分別排在自己的起跑線後方，並使用不同顏色的沙包。
* 每隊準備同色沙包 3 個，尚未投入九宮格的沙包放在該隊起跑線旁。
* 單組模式由隊伍對抗關主代表的「王國守衛」，關主可以比隊伍提早 5 秒開始遊戲。
* 遊戲開始前，關主須示範「跑到投擲線、停在線後投擲、判定結果、返回隊伍交棒」的完整流程。

### 遊戲規則

1. 遊戲開始後，每隊第一位玩家拿取一個己方沙包，跑到投擲線後方進行投擲。玩家不得踩線或跨越投擲線，違者該次投擲無效。
2. 沙包落入尚未被占領的九宮格內，即視為成功占領該格，沙包留在格中，不得任意移動。
3. 出現下列情況時，該次投擲無效，投擲者必須自行將沙包撿回：

   * 沙包完全落在九宮格外。
   * 沙包落在已經有其他沙包的格子。
   * 沙包壓在線上，且關主無法明確判定屬於哪一格。
   * 玩家踩線、跨線或故意以沙包撞擊其他沙包。

   投擲無效後，該名玩家不能在原地再次投擲，必須拿著沙包返回隊伍，交由下一位玩家重新挑戰。

4. 玩家完成有效投擲，或將投失的沙包撿回後，必須跑回起跑線並拍下一位隊員的手，下一位玩家才可以出發。提前出發者須返回起跑線重新開始。

5. 若兩隊的沙包幾乎同時落入同一個空格，以「先完全落定在格內」的沙包取得該格；若關主無法判斷先後，則兩個沙包都取回，該格維持空格。

6. 每隊最多只能有 3 個沙包留在九宮格中。當一隊的 3 個沙包都已成功入格，但尚未完成三連線時，下一位玩家必須：

   * 先跑到九宮格取回一個己方沙包。
   * 再回到投擲線後方。
   * 將該沙包重新投向另一個空格。

   玩家不得直接用手將沙包從原本的格子移到另一格。

7. 任一隊率先讓自己的 3 個沙包形成橫向、直向或斜向三連線，即立即獲勝。

8. 每局限時 2 分鐘。時間到時若雙方都沒有完成三連線，先比較各隊形成的「二連線」數量，數量較多者獲勝。

9. 若二連線數量仍相同，進入驟死賽。雙方各派一名玩家，在清空九宮格後輪流投擲，率先將沙包成功投入中央格的隊伍獲勝。

### 單組模式

1. 單組模式由參賽隊伍對抗關主代表的「王國守衛」。

2. 遊戲倒數開始後，關主先行出發並獲得 **5 秒優先時間**；5 秒倒數結束後，參賽隊伍才可以派出第一位玩家開始投擲。

3. 關主遵守相同的投擲、撿回及移動規則。

### 兩組競爭模式

1. 兩隊同時從各自的起跑線開始遊戲，不設先後手。

2. 每隊隊員依照固定順序接力，不得連續由同一人投擲；所有隊員至少須輪流完成一次後，才可重新由第一位開始。

3. 兩隊共用同一個九宮格，率先以己方 3 個沙包完成三連線的隊伍獲勝。

4. 若時間到仍無隊伍完成三連線，依二連線數量及驟死賽規則決定勝負。

### 遊戲變化

| 模式   | 說明                                            |
| :--- | :-------------------------------------------- |
| 單組闖關 | 隊伍對抗關主代表的王國守衛；關主先開始 5 秒，隊伍在 2 分鐘內搶先完成三連線即闖關成功 |
| 兩組競爭 | 兩隊同時進行接力投擲，共用同一個九宮格，率先完成橫、直或斜三連線者獲勝           |

### 關主注意事項

* 沙包必須以拋投方式投入九宮格，不得用力砸向地面或故意撞開其他沙包。
* 若沙包只有部分壓在格內，以沙包主要面積所在的格子為準；無法判定時視為投擲無效。
* 關主應站在九宮格側邊，避免阻擋玩家投擲，同時負責判斷沙包落點及三連線。
* 玩家撿沙包時可能進入九宮格附近，關主應暫時阻止另一方投擲，等玩家離開危險區域後再繼續。
* 若兩隊人數不同，較少人的隊伍可以較早重複輪替，但仍須維持固定順序。
* 投擲距離應在遊戲開始前測試，避免距離太遠導致所有隊伍持續投失，或距離太近使遊戲失去挑戰性。


### 遊戲獎勵

> 旗子中的「他國」是指其他隊伍的旗子，由關主從備用旗子中隨機發放。本關僅設有「單組闖關」及「兩組競爭」兩種模式。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
| :--- | :- | :-------- | :- |
| 單組闖關 | 成功 | 3 本＋1 他   | 25 |
| 單組闖關 | 失敗 | 2 本＋0 他   | 10 |
| 兩組競爭 | 勝隊 | 3 本＋2 他   | 30 |
| 兩組競爭 | 敗隊 | 1 本＋1 他   | 10 |

> 兩組競爭若限時結束時均未完成三連線，先比較二連線數量；若仍相同，進入中央格投擲驟死賽，因此不另外設置平手獎勵。

**Bonus（星石）**

* **「軍師 Bonus」**：遊戲開始前給予隊伍 10 秒討論投擲順序與目標格，且每位隊員都至少完成一次投擲並最終獲勝，全隊額外獲得 **＋2 星石**。
* **「神射手 Bonus」**：隊伍連續三次投擲皆成功進入有效空格，且中間沒有投失或踩線，全隊額外獲得 **＋2 星石**。
* **「逆轉 Bonus」**：對手已取得兩格連線、距離獲勝只差一格時，隊伍成功阻止對手並完成逆轉三連線，全隊額外獲得 **＋2 星石**。
* 每隊單局最多取得 **4 顆 Bonus 星石**，避免額外獎勵影響整體關卡平衡。

### 特別備註（關主注意事項）

* 沙包必須從投擲線後方投出；玩家踩線或跨越投擲線時，該次投擲視為無效。
* 沙包必須明確落在九宮格內才算成功。若沙包壓在線上，以沙包主要面積所在的格子判定；無法明確判斷時，視為投擲無效。
* 沙包投到九宮格外、已被占領的格子，或因碰撞而無法判定落點時，投擲者必須自行撿回沙包，再返回隊伍交棒。
* 玩家投失後不得站在投擲線原地重投，必須將沙包帶回起跑線，交由下一位隊員重新挑戰。
* 不得故意用沙包撞擊或打飛其他隊伍的沙包。若因正常投擲意外碰到其他沙包，原有沙包放回碰撞前的位置，新投出的沙包視為無效。
* 當玩家進入九宮格附近撿拾沙包時，關主應暫停另一隊投擲，待玩家離開投擲區後再宣布繼續，以免被沙包擊中。
* 玩家必須完成投擲或撿回沙包、跑回起跑線並拍到下一位隊員的手後，下一位才可以出發。
* 跑動以快走或小跑為主，禁止推擠、阻擋對方路線或故意干擾對方投擲；若場地濕滑，統一改為快走。
* 兩組競爭時，兩隊應使用不同顏色的沙包，並盡量設置分開的跑動路線與投擲位置，避免發生碰撞。
* 若兩隊人數不同，人數較少的隊伍可以較早重複輪替，但所有隊員皆須依固定順序出場。
* 單組模式中，關主可以比參賽隊伍提早 **5 秒**開始，但關主也必須遵守相同的投擲、撿回及移動沙包規則。
* 關主應站在九宮格側邊，負責判定沙包落點、投擲是否有效及是否形成三連線，不得站在玩家的正前方。

**違規處理**

* 第一次違規：關主口頭警告，該次投擲無效，沙包須取回。
* 第二次違規：該名玩家返回隊伍，直接失去該回合投擲機會。
* 第三次或出現故意推擠、阻擋、危險投擲等重大違規：關主可判該隊失去一個已占領的格子；若情況嚴重，可直接判定該隊闖關失敗。

<a id="關卡二精靈森域禁語回聲林"></a>
## 關卡二：精靈森域 — 禁語回聲林

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 精靈森域 |
| **需要道具** | 題目卡 20–30 張、計時器、記分板、分組色貼或號碼牌 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **以少勝多（✦特殊規則）**：場面上己方旗子**最少**（至少 1 面）的隊伍佔領。插旗階段可用己方旗子 1:1 替換場面上任意一面旗子（含己方），被換下旗子回到持有者手中。並列最少時以旗子總數決；仍平手先到先得。|
| **適合人數** | 單組 5–10 人；兩組合作 10–18 人；多組競爭 3–5 組皆可 |
| **建議時間** | 講解 1 分鐘；正式 2–3 輪約 5 分鐘；Bonus 1 分鐘 |
| **禁用方式** | 禁止講話、口型、寫字、直接比數字或注音；只能用肢體動作與表情 |

### 故事背景

精靈森域受到沉默結界保護，進入森林的人不能說話，只能靠動作傳遞訊息。每組都是一支來求取通行證的使節團，必須讓訊息在混亂中被理解，才有資格穿越森林。

### 場地與準備

- 單組模式：同組排成一列，第一位看題目後依序傳動作，最後一位回答。
- 多組模式：各組派出同樣人數排成並列縱隊；每組第一位看不同題目，同時開始傳。
- 若想做「跨組傳遞」版本，將隊伍依 A-B-C-A-B-C 排成大圈，每輪傳給右手邊下一位；猜中者所屬隊伍與出題隊伍都得分。

### 遊戲規則

1. 第一位看題目 5 秒後開始表演給下一位看，每次傳遞最多 10 秒。
2. 看完動作者不能回問、不能討論，只能把自己理解的內容繼續演給下一位。
3. 最後一位有 10 秒回答，可答一次；答案意思接近即可，由關主判定。
4. 多組競爭採 3 輪，每輪換一張題目；每組「最後猜對」得 2 分，「中途有人明顯猜對關鍵字」可得 1 分。
5. 跨組版本計分：成功讓別組猜到自己的題目，出題隊 +1；猜中別組題目，猜題隊 +1，因此總分 = 猜對數＋被猜對數。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 同組排成一列傳遞，3 題中至少 2 題最後答案正確即成功 |
| 兩組合作 | 兩組接成一條長隊，最後答案正確即兩組共同成功 |
| 多組競爭 | 各組並列縱隊，輪數結束後依總分排名；同分比答題速度 |
| 跨組傳遞 | 全員圍大圈，每輪傳給右手邊，猜中算雙方得分 |

### 遊戲獎勵

> 兩組合作模式下，兩組各自獲得同等獎勵。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功 | 2 本 ＋ 1 他 | 20 |
| 單組闖關 | 失敗 | 2 本 ＋ 0 他 | 10 |
| 兩組合作 | 成功（各組） | 3 本 ＋ 1 他 | 25 |
| 兩組合作 | 失敗（各組） | 2 本 ＋ 1 他 | 20 |
| 多組競爭 | 第 1 名 | 3 本 ＋ 1 他 | 25 |
| 多組競爭 | 第 2 名 | 2 本 ＋ 0 他 | 15 |
| 多組競爭 | 其餘 | 1 本 ＋ 1 他 | 10 |

**Bonus（星石）**
- 「古精靈語 Bonus」：最後一輪改成抽象詞，答對的隊伍 **＋3 星石**。
- 「無聲完美 Bonus」：整輪無出聲、口型或寫字，該組 **＋1 星石**。

### 特別備註（關主注意事項）

- 題目難度要混合：食物、動物、職業、電影角色、校園物品各準備一些。
- 關主判定「意思接近」即可，不要卡太細（如「電腦」和「筆電」可視題目決定）。
- 若隊伍人數不同，讓人數多的隊伍輪空幾位，保持每組傳遞人數一致。
- **出現說話、口型或手寫提示：該題 0 分；第二次違規可扣 1 分。**

<a id="關卡三龍族火山龍王試煉場"></a>
## 關卡三：龍族火山 — 龍王試煉場

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 龍族火山 |
| **需要道具** | Boss 血量板（建議白板畫愛心）、血量磁鐵或紙條、計時器、Bonus 卡 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **群礁稱王（✦特殊矩陣 6×6）**：使用 6×6 矩陣（36 格）。計算各隊旗子形成的連通區域數（上下左右相鄰為連通），區域數最多者佔領。並列時以旗子總數決；仍平手先到先得。|
| **適合人數** | 單組 4–10 人；多組合作 2–4 組可一起打 Boss |
| **建議時間** | 講解 1 分鐘；正式挑戰 3 分鐘；結算 30 秒 |
| **核心難度** | 只有猜贏才算成功；平手與猜輸皆視為挑戰失敗。造成傷害需要「連續猜贏兩次」 |

### 故事背景

龍族火山口住著守門龍王，牠不靠蠻力決定通行資格，而是用古老的猜拳儀式測試勇者的膽量。每位挑戰者都要連續看破龍王兩次出拳，才能打碎一層龍鱗護盾。

### 場地與準備

- 單組模式：Boss 初始 HP = 3；隊伍排隊依序挑戰關主。
- 多組合作：所有隊伍共同打同一隻 Boss，Boss HP = 參與組數 × 2 + 2（例如 3 組時 HP = 8）。
- 每次挑戰結束後立刻換下一位，避免同一人一直猜。多組合作時依 A-B-C-A-B-C 順序輪流派人。

### 遊戲規則

1. 挑戰者必須連續猜贏關主兩次，才對 Boss 造成 1 點傷害。
2. 若第一拳或第二拳平手／猜輸，該次挑戰失敗，直接換下一位。
3. 限時 3 分鐘內將 Boss HP 歸零即闖關成功。
4. 每位玩家至少要出場一次；若人數太多，關主可以縮短每次猜拳間隔並用倒數加速。
5. 多組合作仍記錄各組造成的傷害數，作為 Bonus 與貢獻獎勵依據，但主要目標是共同打倒 Boss。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |

| 多組合作 | Boss HP = 組數 × 2 + 2，所有組輪流出人打同一隻王 |
| 多組改競爭（備案） | 各組獨立對關主，計 3 分鐘內造成的傷害數排名 |

### 遊戲獎勵

> 合作模式中各組各自領取同等獎勵。龍王火山的寶藏中夾帶其他王國旗幟，成功討伐者可獲得更多他國旗子。

- 「龍焰暴擊」：同一玩家連續猜贏三次，該組 **＋2 星石**。
- 「全員勇者」：所有人至少上場一次並打倒 Boss，全組 **＋2 星石**。

### 特別備註（關主注意事項）

- 關主出拳要自然，不要讓玩家覺得故意讀心或故意放水。
- 多組合作時，關主需要大聲報 Boss 剩餘 HP，讓氣氛比較燃。
- 如果隊伍太小或運氣太差，可以中途啟用「火山祝福」：HP 減 1 或改成連贏一次造成半點，避免卡太久。

## 關卡四：獸人荒原 — 荒原躲避戰

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 獸人荒原 |
| **需要道具** | 軟式躲避球 1–2 顆、角錐或膠帶邊界、隊伍色帶、計時器 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **潮汐均勢（✦特殊規則）**：世界史更新時先計算全場旗子總數；總數為奇數則本輪無人佔領（領地中立）；總數為偶數則旗子最多的隊伍佔領。並列先到先得。|
| **適合人數** | 單組 5–10 人；兩組競爭各 5–10 人；多組競爭 3 組可採混戰或輪戰 |
| **建議時間** | 安全提醒 1 分鐘；正式每局 3 分鐘；決鬥或結算 1 分鐘 |
| **安全規則** | 禁止瞄頭、禁止高速近距離砸人、禁止推擠；頭部被打不算出局，投擲方警告 |

### 故事背景

獸人荒原的部族不靠口頭辯論決定通行權，而是用古老的躲避戰分出勇士。能在荒原球戰中保護隊友、閃避攻擊並留到最後的隊伍，才會獲得獸人部族的戰旗。

### 場地與準備

- 兩組競爭：場地分成左右兩半，各隊在自己的半場活動。
- 多組競爭：若場地夠大，分成 3–4 個區域混戰；若場地不夠，採輪戰積分制。
- 單組模式：改為「獸人訓練場」，隊伍派 2–3 位閃避者，其他人或關主在安全距離外投球，挑戰 60 秒存活。

### 遊戲規則

1. 正式局限時 3 分鐘，開始時球放中線或由關主拋球。
2. 球直接擊中身體且未先落地，該玩家出局；若被接住，投球者出局，接球隊可復活 1 位最早出局的隊友。
3. 球落地後再碰到人不算；碰到頭部不算，且投球方被警告。
4. 多組混戰時，可以攻擊任一隊，但不能跨出自己的區域太多；出界接球或投球皆無效。
5. 單組訓練場：60 秒內保留至少一半閃避者即成功；若全員存活視為完美成功。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組訓練場 | 派 2–3 閃避者，關主投球，60 秒存活過半即成功 |
| 兩組競爭 | 左右半場對打，3 分鐘比剩餘人數 |
| 多組混戰 | 各組佔一區域，可攻擊任意隊，依出局順序排名 |

### 遊戲獎勵

> 戰敗或失敗時，敵方旗幟散落在戰場上，因此敗者會拿到他國旗子。

| 遊玩模式 | 結果 | 旗子（本國＋他國） | 星石 |
| :---- | :---- | :---- | :---- |
| 單組闖關 | 成功 | 2 本 ＋ 0 他 | 15 |
| 單組闖關 | 失敗 | 1 本 ＋ 1 他 | 10 |
| 兩組競爭 | 勝隊 | 3 本 ＋ 0 他 | 20 |
| 兩組競爭 | 敗隊 | 1 本 ＋ 1 他 | 10 |
| 兩組競爭 | 平手（各隊） | 2 本 ＋ 0 他 | 15 |
| 多組競爭 | 第 1 名 | 3 本 ＋ 0 他 | 20 |
| 多組競爭 | 第 2 名 | 2 本 ＋ 0 他 | 15 |
| 多組競爭 | 其餘 | 1 本 ＋ 1 他 | 10 |

**Bonus（星石）**
- 「獸人戰旗 Bonus」：全局無頭部攻擊、推擠或爭議動作，全隊 **＋1 星石**。
- 「神接殺 Bonus」：成功接殺並復活隊友，該隊 **＋1 星石**；每局最多 ＋2。

### 特別備註（關主注意事項）

- 一定要用軟球，且先講安全規則；此關安全優先於勝負。
- 若隊伍人數不同，少人隊可多 1 次復活，或多人隊先派同樣人數上場。
- 西部牛仔決鬥（平手加賽）：各派 1 人背對背走 5 步，猜拳決定先攻，轉身投球；沒打到就交換球權，每輪平手步數遞減為 4、3、2。
- **若有人危險動作，關主可直接暫停，警告後重開或判該隊該回合失分。**
