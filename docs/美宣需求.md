## 一、最終大戰靜態網頁素材

### 1.1 網頁概述

設計需求：靜態網頁顯示在 103 教室投影幕，用來呈現最終大戰（WWW）的即時戰況。

- 靜態畫面會由一個固定背景 + 12 個島嶼 + 4 個區域組成
- 動態素材包含旗子 + 小帆船
- 希望能提供素材的色碼（主要用到的顏色就好不用到全部，當然有全部也可以），在做一些調整會比較方便。
- 分開的素材的話幫我去背，感謝～
- 大概想像是一張大地圖，大家在上面打架，船會在不同島嶼之間開來開去代表進攻。
- **重點是地圖要放得下 12 個島嶼 + 4 個區域 + 讓船在上面跑** 這些考慮到在 103 要看，所以不用太精緻，遠看有 feel 就好。
- 基於上一點，畫完可以稍微擺擺看。如果有特別擺過覺得很讚的話也可以直接將合體的靜態畫面的部分給我。沒有的話可以給我分開的素材我會自己排版。

## 二、實體道具美術設計

### 2.1 道具卡（49 張）

**卡面基本規格**

| 項目 | 說明 |
| :---- | :---- |
| **卡面尺寸** | 建議 A6（105×148mm），或標準遊戲卡 63×88mm |
| **卡面文字** | 道具名稱 + 稀有度標記（N/R/SR/SSR）+ 效果說明 |
| **卡面插畫** | Optional：每種道具卡一個小圖示 |

- [道具卡文字與等級](https://hackmd.io/@vzNDFDASTzyhGfQ5x3q3DQ/Hk6pPnfGzg)

**各稀有度邊框設計要求**

> 原則：越稀有邊框越複雜、越華麗。美宣無需會特效，用顏色和花紋粗細區分即可。

**N 卡（普通）**
- 邊框粗細：細（2–3mm 即可）
- 顏色：純灰色（建議 `#9E9E9E`）
- 樣式：單線邊框，四個角直角
- Optional：角落加一個小菱形裝飾

**R 卡（稀有）**
- 邊框粗細：中等（4–5mm）
- 顏色：金黃色（建議 `#D4AF37`）
- 樣式：雙線邊框（外粗內細，外粗 4mm、內細 1mm，兩條線之間留 2mm 空隙）
- 四個角加一個小菱形或正方形裝飾角
- Optional：邊框內側加細點線紋（像裝飾性花邊，等距小點）

**SR 卡（超稀有）**
- 邊框粗細：中粗（6–7mm）
- 顏色：藍紫漸層——從卡片上緣藍色（`#3F51B5`）漸層到下緣紫色（`#9C27B0`）
- 樣式：單線邊框，但四個角有角括弧裝飾（`「 」` 形狀，每角一個）
- Optional：邊框左右兩側各加 2–3 個小星星（★ 符號）

**SSR 卡（最高稀有）**
- 邊框粗細：粗（8–10mm）
- 顏色：金色（`#FFD700`）為主；若製作時間足夠，可從金色漸層到橙紅（`#FF6F00`）
- 樣式：雙線邊框（外粗金色 8mm、內細白色 1mm），四個角有皇冠形或交叉劍形裝飾角
- 頂部邊框中央加「SSR」小字標籤（可用色框圈住）
- Optional：邊框外側加細光芒線（8 條短線從角向外放射，像太陽光芒），若嫌複雜可省略
