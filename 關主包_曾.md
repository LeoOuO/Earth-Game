## 關卡一：巨人山丘 — 肢體石陣 {#關卡一巨人山丘肢體石陣}

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 巨人山丘 |
| **需要道具** | 巧拼墊或瑜伽墊 2–4 組、題目卡、計時器、哨子、地面邊界膠帶、記分表 |
| **需要關主人數** | 2 人 |
| **佔領規則** | **城牆防禦**：外圍 12 格（中央 4 格 (1,1)(1,2)(2,1)(2,2) 不計）旗子**最多**的隊伍佔領。平手先到先得。|
| **適合人數** | 單組 4–10 人；兩組競爭各 4–10 人；多組競爭 3–5 組皆可。人數差距大時，取同樣人數上場，其餘隊員可輪替 |
| **建議時間** | 講解 1 分鐘；正式 5 題約 5 分鐘；結算 1 分鐘 |
| **安全規則** | 禁止踩人、壓脖子、跳到隊友身上；若姿勢看起來危險，關主可要求重做或換題 |

### 故事背景

巨人山丘上有一座會辨識身體姿勢的古老石陣。巨人族相信，只有能把隊伍身體部位拼成正確圖形的旅人，才懂得合作與承重，因此每一題都是巨人族給外來者的通行試煉。

### 場地與準備

- 每組在自己的巧拼墊或指定區域內挑戰；未參與該題的人站到區域外，不計入觸地部位。
- 關主將題目分成簡單、中等、困難三種，題目越難，該題基礎金幣越高。
- 每題開始前給 5 秒讀題與討論；喊「開始」後才可以移動，完成後喊「完成」並定格 3 秒。

### 遊戲規則

1. 關主念出題目，例如「三隻手、一個頭、兩隻腳」，各組必須讓區域內剛好只有指定數量的身體部位接觸巧拼墊。
2. 「手」只算手掌或手背，「腳」只算腳底或腳背，「頭」只算頭部碰墊；手肘、膝蓋、屁股不算手或腳，除非題目明確指定。
3. 兩組 PK 時，兩隊同時做同一題；先完成且通過驗證者記錄完成秒數，另一隊可繼續做到時間上限。
4. 若兩組都完成，完成秒數差 k 秒，先完成的組別額外獲得 k 枚金幣；建議差距獎上限先設 5 枚，現場可再平衡。
5. 多組競爭時，各組同時挑戰同一題，依完成時間排名；若場地不夠，可分批挑戰並用秒數排行。
6. 單組模式由隊伍在限時內連續挑戰 5 題，累積金幣達標即闖關成功。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 5 題累積達 12 枚金幣以上算成功 |
| 兩組競爭 | 兩隊同題 PK，秒數差 k 決定差距獎；整關總金幣高者再 +3 |
| 多組競爭 | 各組同時挑戰，依完成時間排名，總分第一再 +3 |

**題目難度與基礎金幣**

| 難度 | 基礎金幣 | 題目範例 |
| :---- | :---- | :---- |
| 簡單 | 2 | 四隻腳兩隻手；兩隻手一個頭；三隻腳一隻手 |
| 中等 | 4 | 三隻手一個頭兩隻腳；兩隻腳兩個膝蓋一隻手 |
| 困難 | 6 | 一個頭三隻手一隻腳；兩個膝蓋一個肩膀兩隻手 |
| Bonus | 8 | 全員至少一人碰墊，但指定部位仍要剛好正確 |

### 遊戲獎勵

| 遊玩模式 | 成功／勝利 | 失敗／其餘名次 |
| :---- | :---- | :---- |
| 單組闖關 | 完成每題拿基礎金幣；5 題累積 12 枚以上算成功，成功額外 +2 | 未達 12 枚給實得金幣；若低於 4 枚，保底 4 |
| 兩組競爭 | 每題先完成者拿「基礎金幣 + min(k,5)」；整關總分高者再 +3 | 後完成者拿基礎金幣；整關落敗保底 5；平手各 +1 |
| 多組競爭 | 每題第一名拿「基礎 +3」、第二名「基礎 +1」、其餘完成者拿基礎；總分第一再 +3 | 完全未完成者 4；整關第三後保底 5 |

**Bonus**
- 「巨人完美定格」：完成後 3 秒內完全不晃、不倒、不多碰地，該題 +1。
- 「困難題 Bonus」：自願抽困難題並成功，該題基礎金幣 +2；失敗不倒扣。

### 特別備註（關主注意事項）

- 關主驗證時要先看「有沒有多出不該碰地的部位」，再看「指定部位數量是否足夠」。
- 題目不要一開始就太難，建議 2 題簡單、2 題中等、1 題困難，避免卡關時間太久。
- 人數差距大時，採「同人數上場」，例如每題各派 5 人，其他人輪替，避免人多隊太有利。
- 每題時間上限 60 秒；時間到仍未完成者該題 0 枚，但可繼續下一題。
- **若出現危險姿勢、故意推人、踩人，關主可判該題失敗；嚴重者該組本關最低只給參與保底。**

## 關卡二：侏儒劇場 — 節拍回聲圈 {#關卡二侏儒劇場節拍回聲圈}

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 侏儒劇場 |
| **需要道具** | 音樂播放設備、計時器、隊伍色貼、記分板；可準備小球或權杖代表「節拍權」|
| **需要關主人數** | 1 人 |
| **佔領規則** | **舞台正中**：中間兩行（第 1 行 + 第 2 行，共 8 格）旗子**最多**的隊伍佔領。|
| **適合人數** | 單組 6–12 人；兩組競爭各 4–8 人；多組競爭 3–5 組皆可，最推薦多組一起玩 |
| **建議時間** | 講解與試玩 1–2 分鐘；正式 3 輪，每輪 45 秒；結算 1 分鐘 |
| **禁用方式** | 不可提醒隊友、不可用口型偷講答案、不可故意喊太小聲；喊詞與動作都要讓關主看得到 |

### 故事背景

侏儒劇場裡的演員都用「詞語咒語」和「動作印記」辨認彼此。只要音樂一停，舞台上的咒語就會開始亂跳，參賽隊伍必須記住每個人的詞語與動作，才能在混亂的節拍中把表演接下去。

### 場地與準備

- 所有人圍成一圈，多組時盡量依 A-B-C-A-B-C 交錯站位；兩組時依 A-B-A-B 交錯。
- 每位玩家先想一個詞語和一個對應動作，例如「鳳梨」配雙手比尖刺、「火箭」配往上發射。詞語不可重複。
- 正式開始前，全體依序示範自己的詞語與動作一次；關主確認大家大致記得後開始播放音樂。

### 遊戲規則

1. 音樂播放時大家跟拍或原地小動作，音樂停止後，由關主指定第一位或由拿到權杖的人開始。
2. 輪到的人必須在 3 秒內完成：「做自己的動作 + 喊出另一人的詞語 + 做出那個人的動作」。
3. 被喊到的人接續同樣流程，不能喊回上一個人；多組競爭時，原則上不能連續喊同組兩次。
4. 每成功接龍一次，執行者所屬隊伍 +1 分；喊錯詞語、做錯動作、超過 3 秒、喊回上一人，該次不給分並由關主重新指定起點。
5. 正式採 3 輪，每輪 45 秒；每輪可以讓不同人當起點，避免某一隊一直沒有出手機會。
6. 單組模式：改成挑戰連續接龍，60 秒內成功接龍 12 次以上即成功；中途失誤不歸零，但會浪費時間。

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 60 秒內成功接龍 12 次以上即成功 |
| 兩組競爭 | 交錯站位，3 輪後比總分 |
| 多組競爭 | 全員圍圈 A-B-C 交錯，3 輪後依總分排名 |

### 遊戲獎勵

| 遊玩模式 | 成功／勝利 | 失敗／其餘名次 |
| :---- | :---- | :---- |
| 單組闖關 | 60 秒內成功接龍 12 次以上得 8 金幣；15 次以上再 +2 | 未達 12 次得 4 金幣；達 8–11 次給 5 |
| 兩組競爭 | 總分高者 10 金幣；若三輪全勝再 +2 | 落敗隊 5 金幣；平手兩隊各 7 |
| 多組競爭 | 第一名 12、第二名 9、第三名 7；第一名零失誤再 +2 | 其餘每組 5；嚴重違規或干擾者最低 4 |

**Bonus**
- 「劇場無失誤 Bonus」：任一組整輪被喊到時都沒有失誤，該組 +2。
- 「抽象詞 Bonus」：最後一輪改成抽象詞（如「尷尬、勝利、遲到、考爆」），成功被辨識的組別 +2。

### 特別備註（關主注意事項）

- 關主先讓大家試玩 20 秒，否則正式開始會很容易全場不知道要幹嘛。
- 詞語以 2–4 字最好，太長太像句子會讓節奏斷掉；動作要明顯，不要太細。
- 如果人太多，可以每組只派 3–4 人上場，其他人下一輪替換，保持圈不要太大。
- **若有人在場外提醒、口型提示或故意干擾別組，第一次警告，第二次該隊扣 1 分。**
- 若玩家忘記自己的詞語或動作，可在下一輪開始前重新設定，但該輪中途不能更改。

## 關卡三：狐族賭館 — 謊言骰杯 {#關卡三狐族賭館謊言骰杯}

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 狐族賭館 |
| **需要道具** | 每組 5 顆骰子與 1 個骰盅或紙杯、生命籌碼、計時器、叫價紀錄小白板；共需 2 套（兩個關主） |
| **需要關主人數** | 2 人 |
| **佔領規則** | **對角密謀**：兩條對角線格（(0,0)(1,1)(2,2)(3,3) 和 (0,3)(1,2)(2,1)(3,0)，共 8 格）旗子**最多**的隊伍佔領。|
| **適合人數** | 單組 4–10 人；兩組競爭各 4–10 人；多組競爭 3–5 組可採擂台或淘汰賽 |
| **建議時間** | 講解 1 分鐘；試玩 1 局；正式三戰兩勝約 5 分鐘 |
| **基本設定** | 本關不使用「1 點萬用」以降低規則複雜度；若要加難度可把 1 點萬用放到 Bonus |

### 故事背景

狐族賭館不看誰力氣大，而是看誰能在半真半假的骰杯聲中保持冷靜。每一組都是來談判的商隊，必須透過叫價、判斷與抓謊，從狐族手中贏得通行用的金幣。

### 場地與準備

- 每組派 1–3 位代表坐到賭桌前，其餘隊員可在每次輪到自己組別時提供 10 秒討論。
- 每組拿 5 顆骰子，搖完後只能自己組內偷看，不可給其他組看到。
- 兩組 PK 建議採三戰兩勝；多組競爭可採每組 2 條生命，輸一次扣 1 條，生命歸零出局。

### 遊戲規則

1. 第一組喊出一個叫價，格式為「數量 + 點數」，例如「三個 4」代表全場所有骰子中至少有 3 顆是 4 點。
2. 下一組若不開骰，就必須提高叫價：可以增加數量，或在同數量下提高點數（例如「三個 4」後可喊「三個 5」或「四個 2」）。
3. 任一組覺得上一家的叫價不可信，可以喊「開」。所有組翻開骰子，計算該點數實際出現幾顆。
4. 若實際數量大於或等於上一家的叫價，上一家成功，喊開者輸；若實際數量不足，喊開者成功，上一家輸。
5. 兩組 PK 時，輸方失去 1 條生命，先讓對方失去 2 條生命者勝；多組時依生命與出局順序排名。
6. 單組模式：隊伍對關主，5 局中贏 3 局即成功；關主可固定用 5 顆骰子。

**叫價示例**

| 目前叫價 | 下一家可以喊 | 下一家不能喊 |
| :---- | :---- | :---- |
| 三個 4 | 三個 5、三個 6、四個 1 | 三個 3、兩個 6 |
| 五個 2 | 五個 3、六個 1 | 四個 6、五個 1 |

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 隊伍對關主，5 局贏 3 局即成功 |
| 兩組競爭 | 三戰兩勝制 |
| 多組競爭 | 每組 2 條生命，輸一次扣一條，最後存活者第一名 |

### 遊戲獎勵

| 遊玩模式 | 成功／勝利 | 失敗／其餘名次 |
| :---- | :---- | :---- |
| 單組闖關 | 對關主 5 局贏 3 局得 8 金幣；贏 4 局以上再 +2 | 未達 3 勝得 4 金幣；若 2 勝給 5 |
| 兩組競爭 | 勝隊 10 金幣；若直落二再 +2 | 敗隊 5 金幣；若打滿三局惜敗給 6 |
| 多組競爭 | 第一名 12、第二名 9、第三名 7 | 其餘每組 5；首輪出局但有贏過一局給 6 |

**Bonus**
- 「神抓 Bonus」：喊開後剛好差 1 顆就抓成功，喊開組 +2。
- 「萬用 1 Bonus」：Bonus 局可設定 1 點為萬用；Bonus 局勝利的組別 +3，但規則要先講清楚。

### 特別備註（關主注意事項）

- 這關是遊戲代幣，不能真的賭錢，也不要讓玩家用私人財物當籌碼。
- 關主最好在旁邊記錄上一個叫價，避免有人忘記目前喊到多少。
- 若參與者完全不會吹牛，先示範一局「叫價、加價、喊開、翻骰」會比較快進入狀況。
- **偷看別組骰子、翻骰前改口、場外大聲爆雷：第一次警告並重搖，第二次該局判負。**

## 關卡四：機械王國 — 四色旗令台 {#關卡四機械王國四色旗令台}

| 欄位 | 內容 |
| :---- | :---- |
| **所屬領地** | 機械王國 |
| **需要道具** | 每組紅、綠、藍、黃旗各 1 面（若只有三色，也可只用紅、綠、藍）；題目單、計分板 |
| **需要關主人數** | 1 人 |
| **佔領規則** | **精密連鎖**：水平相鄰旗子對**最多**的隊伍佔領。（同行左右緊鄰算 1 對，全場最多 12 對）|
| **適合人數** | 單組 4–10 人；兩組競爭各派 2 人上台；多組競爭 3–6 組同時進行 |
| **建議時間** | 講解 1 分鐘；正式 10 回合約 3 分鐘；加賽 1 分鐘 |
| **核心判定** | 一回合內所有被指令提到的旗子都做對，該組才算該回合正確；對幾回合就先拿幾枚基礎金幣 |

### 故事背景

機械王國的城門由四色訊號旗控制，守門機器人只聽懂精準的升降指令。旅人必須派出旗手，在一連串「升起來、不要升、降下來、不要降」的混亂口令中保持正確狀態，才會被判定為合格通行。

### 場地與準備

- 每組派 2 人上台：一人拿紅旗與綠旗，另一人拿藍旗與黃旗；若只用三色，黃旗可取消。
- 所有旗子一開始都放下。每回合開始前，關主確認各組旗子狀態，然後念出完整指令。
- 隊友與台下不可出聲提示；上台兩人可以用眼神配合，但不能互相講話。

### 遊戲規則

1. 關主每回合念一串指令，例如「紅旗升起來，綠旗不要升，藍旗降下來」。
2. 參賽者必須等關主念完「開始」後才動作；搶先動作該回合不計分。
3. 「升起來」= 舉高；「降下來」= 放下；「不要升」與「不要降」= 保持原本狀態，不可被誘導改變。
4. 每回合給 2 秒完成動作，關主確認所有被點名的旗子狀態都正確，該組該回合 +1 金幣。
5. 正式共 10 回合；每組答對幾回合就先拿幾枚基礎金幣。
6. 兩組或多組競爭時，10 回合後比較正確回合數；若同分，進入 3 題加賽，誰先出錯誰排名較後。

**題目範例**

| 回合 | 口令示例 | 設計目的 |
| :---- | :---- | :---- |
| 1 | 紅旗升起來，綠旗不要升。 | 確認升起與保持不動 |
| 2 | 藍旗升起來，紅旗降下來。 | 練習雙色切換 |
| 3 | 綠旗升起來，藍旗不要降。 | 加入「不要降」干擾 |
| 4–10 | 紅旗不要升，藍旗降下來，黃旗升起來。 | 提高混亂度與反應難度 |

### 遊戲變化

| 模式 | 說明 |
| :---- | :---- |
| 單組闖關 | 10 回合，答對 7 以上算成功 |
| 兩組競爭 | 各隊先拿「答對回合數」金幣；勝隊額外 +3 |
| 多組競爭 | 各隊先拿「答對回合數」金幣；第一名 +3、第二名 +2、第三名 +1 |

### 遊戲獎勵

| 遊玩模式 | 成功／勝利 | 失敗／其餘名次 |
| :---- | :---- | :---- |
| 單組闖關 | 基礎金幣 = 答對回合數；答對 7 回合以上算成功並額外 +2，滿分再 +2 | 答對 0–3 回合保底 4；答對 4–6 回合給實得金幣 |
| 兩組競爭 | 各隊先拿「答對回合數」金幣；勝隊額外 +3，若滿分再 +2 | 敗隊拿實得金幣，最低 5；平手兩隊各額外 +1 |
| 多組競爭 | 各隊先拿「答對回合數」金幣；第一名 +3、第二名 +2、第三名 +1 | 其餘拿實得金幣，最低 5；嚴重違規者最低 4 |

**Bonus**
- 「零失誤旗手 Bonus」：10 回合全對，該組 +3。
- 「地獄口令 Bonus」：追加 3 題，每題包含至少 4 個指令；答對 2 題以上 +2，3 題全對 +3。

### 特別備註（關主注意事項）

- 關主念題速度要由慢到快，前 3 題當暖身，中間開始加入「不要升／不要降」干擾。
- 指令每回合最多 3 個動作，避免玩家完全聽不懂；Bonus 才使用 4 個以上動作。
- 同一組不同回合可以換人上台，但每回合開始後不可中途換人。
- **若某組上台兩人互相講話、台下提示或故意干擾別組，第一次該回合 0 分，第二次扣 1 分。**
- 若道具旗掉落，該回合可重來一次；若是玩家自己甩太大力造成，該回合 0 分。
