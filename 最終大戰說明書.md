## 一、賽前準備

- **初始兵力**：等於大地遊戲期間四次世界史更新（U1–U4）**累積的兵力總和**。詳細兵力獲得規則見[總規則說明書](總規則說明書.md)第六節。
- **初始佔領**：大地遊戲結束時（15:10），誰佔領哪個領地，最終大戰就從哪個領地持有兵力開始。
- **未持有領地**的隊伍從中立區域出發。

## 三、回合操作

每個回合（7 分鐘）最多可執行 **3 次操作**，所有操作**平行執行**（同一回合內的操作互不阻擋）。

### 操作一：移動兵力

```
moving(S, E, n)
```

| 參數 | 說明 |
| :---- | :---- |
| S | 出發區域（必須為己方領地、中立區域或資源點）|
| E | 目的區域（同上限制）|
| n | 移動兵力數（正整數，不超過 S 當前兵力）|

- **不可**移動至他國領地。

---

### 操作二：進攻

```
attack(S, E, n)
```

| 參數 | 說明 |
| :---- | :---- |
| S | 出發區域（必須為己方領地、中立區域或資源點）|
| E | 目標（必須為**他國領地**）|
| n | 派遣兵力數 |

- 若 E 為中立區域或資源點，視為違反世界法，S 損失 n 兵力（自殺）。
- 可以多個 attack 從不同區域同時進攻同一個 E。

---

### 操作三：聯盟（防禦型）

```
union(S, E, P, n)
```

| 情況 | 說明 |
| :---- | :---- |
| E 為 P 國領地 | 請求加入聯盟，在 P 同意時，將 S 的 n 兵力駐守於 E |
| E 為我國領地 | 接受 P 國的聯盟邀請，同意 P 的兵力駐守於 E |

**聯盟成立條件（缺一不可）：**
1. 雙方都發出 union
2. 雙方設定的 n **嚴格相同**

---

### 操作四：聯合進攻

```
union_attack(S, E, P, n)
```

| 參數 | 說明 |
| :---- | :---- |
| S | 出發區域 |
| E | 目標領地（必須為非我國、非 P 國的他國領地）|
| P | 聯合進攻國（可為複數）|
| n | 派遣兵力數 |

**合法聯盟條件（最大嚴格覆蓋原則）：**
- 所有參與國互相在 P 列表中指名對方，才構成完整聯盟。
- 未被互相指名者，獨立計算為一個單獨進攻單位。

> **範例 1（三國完整聯盟）：**
> - P1: `union_attack(A, [P2,P3], E, n1)`
> - P2: `union_attack(A, [P1,P3], E, n2)`
> - P3: `union_attack(A, [P1,P2], E, n3)`
> - 結果：P1+P2+P3 以 n1+n2+n3 合力進攻 E
>
> **範例 2（不完整聯盟，P3 只指名 P1）：**
> - P1: `union_attack(A, [P2,P3], E, n1)`
> - P2: `union_attack(A, [P1,P3], E, n2)`
> - P3: `union_attack(A, [P1], E, n3)`
> - 結果：P1+P3 以 n1+n3 進攻 E；P2 獨立以 n2 進攻 E

## 四、回合結算規則

### 1. 領地重新結算領主

若任一國家對某領地發起進攻（attack 或 union_attack），則：

- **佔領方**：派遣/駐守兵力**最大**的單位（單國或聯盟）成為新領主。
- **佔領方獲得兵力**：自身兵力 + 20% × 各失敗方派遣兵力
- **失敗方**：損失全部派遣兵力。

> **範例：**
> - P1（原佔領者）：100 兵力駐守
> - P2：`attack(A, E, 99)`
> - P3：`attack(A, E, 99)`
> - 結果：P1 維持佔領，最終兵力 = 100 + ⌊99×20%⌋ + ⌊99×20%⌋ = **138**

### 2. 領地產出（國力）

- 若結算時只有 A 國佔領：A 國國力 +X（X 為該領地每回合產出）
- 若有佔領國 A 與駐守國（含 union 聯盟）：
  - A 國獲得：½X + ½X × A的兵力比例
  - B 國（駐守）獲得：½X × B的兵力比例
  - 以此類推

### 3. 中立區域結算

- 駐守 n 兵力的國家：兵力變為 n × 1.5（無條件捨去小數）
- **0 兵力救濟**：兵力為 0 的國家每回合額外獲得 **1000 兵力**（含強制移入中立區域者）

### 4. 資源點結算

- 依各國駐守兵力比例分配資源（含小數無條件捨去）

## 五、最終排名與懲罰獎勵

- **排名依據**：3 回合結算結束後的**國力總量**
- **第一、二名**：印製國王帽子，到台上戴帽子
- **倒數第一、二名**：到台上跳 67

## 六、操作衝突裁決

若一個回合提交的操作**所需兵力總和超過當前可用兵力**：

1. 所有操作視為**無效**（不執行任何移動或攻擊）
2. 該小隊本回合兵力強制降為 **0**
3. 強制移入**中立區域**，並觸發 0 兵力救濟（每回合 +1000 兵力）

> **範例（衝突觸發）**：
> - A 國當前兵力 100
> - 提交：`moving(A, B, 50)` + `attack(A, C, 80)` → 所需 130 > 100
> - 結果：兩個操作全部無效，A 國兵力歸零，強制移入中立區域

## 七、操作範例

### 範例：平行操作的重要性

若同一回合執行以下操作：
```
moving(A, B, 50)
attack(A, C, 80)
```
由於操作**平行執行**，兩個指令都視為從回合開始時的原始兵力狀態出發。若兵力足夠（例如共 150），則兩個操作均正常執行；若兵力不足（例如只有 100），則觸發衝突裁決，全部操作無效並移入中立區域。
